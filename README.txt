# SLA_Printer
Exhibits the working of a basic SLA 3D printer
AUTHOR: Astha Prasad
EMAIL: asthaprasad93@gmail.com
PURPOSE: To write a C++ program that mimics the working of a simple SLA 3D printer, given the necessary print settings and layer images

DEPENDENCIES:
Tested with openCV 2.4.13

BUILDING THE CODE:
cd SLA_Printer
mkdir build
cd build
cmake ..
make

RUNNING THE CODE:
cd SLA_Printer
./build/main "path_to_Print_Bundle"
